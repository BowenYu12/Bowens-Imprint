Effective Date: July 8, 2026

Privacy Policy for Usage

Usage is designed to help users understand and track their usage-related information in a simple and visual way.

Information We Collect

Usage does not collect, sell, or share personal information with third parties.

The app may store usage-related settings, display preferences, history records, and app configuration data locally on the user's device. This data is used only to provide the app's core features.

Network and API Configuration

Usage may connect to supported online services only when required for app functionality. Any authentication or configuration information is used only to enable the user-requested features and is not sold or shared with advertisers.

Local Storage

Some information may be stored locally on the user's device or in shared app storage for widgets and app functionality. Users may remove the app from their device to delete locally stored app data.

Third-Party Services

Usage may interact with Apple system services and supported API services only as necessary for app functionality. Usage does not use third-party advertising trackers.

Children's Privacy

Usage is not directed to children under the age of 13. We do not knowingly collect personal information from children.

中文隐私政策

生效日期：2026年7月8日

Usage 是一款用于帮助用户理解和查看使用量相关信息的轻量工具。

我们收集的信息

Usage 不会收集、出售或向第三方分享用户的个人信息。

App 可能会在用户设备本地保存使用量相关设置、显示偏好、历史记录和应用配置数据。这些数据仅用于提供 App 的核心功能。

网络与 API 配置

Usage 可能会在功能需要时连接到受支持的在线服务。任何认证或配置信息仅用于用户请求的功能，不会出售或分享给广告商。

本地存储

部分信息可能会保存在用户设备本地，或用于小组件和 App 功能的共享存储中。用户可以通过删除 App 来移除本地保存的数据。

第三方服务

Usage 可能会在必要时与 Apple 系统服务或受支持的 API 服务交互。Usage 不使用第三方广告追踪器。

儿童隐私

Usage 并非面向 13 岁以下儿童。我们不会有意收集儿童的个人信息。
