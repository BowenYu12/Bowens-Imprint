# Bowen's Imprint / 博文之痕

Bowen's Imprint is a static personal portfolio and product website for software, games, writing, photography, design experiments, and App Store support pages.

## Structure

- `index.html` - homepage
- `privacy/usage/index.html` - Usage privacy policy
- `support/usage/index.html` - Usage support page
- `assets/` - shared CSS and JavaScript

## Deploy with GitHub Pages

1. Create a public GitHub repository.
2. Upload these files.
3. Go to Settings -> Pages.
4. Select Branch: `main`.
5. Select folder: `root`.
6. Save.
7. The website will be available at:
   `https://USERNAME.github.io/REPOSITORY_NAME/`

The Usage privacy policy URL will become:

`https://USERNAME.github.io/REPOSITORY_NAME/privacy/usage/`
