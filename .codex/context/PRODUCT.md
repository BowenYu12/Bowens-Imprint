# Product memory

## Website

- Name: **Bowen's Imprint / 博文之痕**.
- Type: public personal portfolio, archive, and product-support website.
- Hosting intent: static GitHub Pages site served from the repository root.
- Languages: homepage and Usage policy/support content include English and Chinese.

## Public purpose

- Collect software, games, books/writing, photography, and design experiments.
- Present finished work, prototypes, and future releases in one personal archive.
- Provide durable public privacy-policy and support URLs for released apps.
- Current featured product: the Usage iOS app.

## Experience

- Light-first editorial presentation with a neutral gray field, compact navigation,
  restrained metadata, and a high-contrast bilingual masthead.
- Homepage is structured as a project index: masthead, four numbered image-led
  modules in a desktop grid/mobile list, and a compact footer.
- Project names and metadata appear on mouse hover, keyboard focus, or mobile tap.
- Current modules are Usage, Thunder Fighter, Ten Weapons, and Photography.
- Usage has dedicated, bilingual privacy and support pages using the same visual
  system.
- Contact address is published on the site and in support/policy content.

## Media status

- Usage artwork is based on the current SwiftUI dashboard structure.
- Thunder Fighter artwork uses the local game's HUD language and equipment icons.
- Ten Weapons artwork is based on the local manuscript and chapter count.
- Photography currently uses a clearly described neutral placeholder and should be
  replaced with Bowen's own selected photograph when one is available.

## Trust boundaries

- No backend, account system, analytics, advertising, or build framework is present
  in this repository.
- Public legal/support wording must stay aligned with actual app behavior.
- Do not publish private drafts, credentials, or personal data that is not already
  intentionally public.
