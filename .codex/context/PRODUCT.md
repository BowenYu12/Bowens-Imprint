# Product memory

## Website

- Name: **Bowen's Imprint / 博文之痕**.
- Type: public personal portfolio, archive, and product-support website.
- Hosting intent: static GitHub Pages site served from the repository root.
- Languages: homepage and Usage policy/support content include English and Chinese.

## Public purpose

- Collect software, games, books/writing, photography, and design experiments.
- Present finished work, prototypes, and future releases in one personal archive.
- Provide durable public privacy-policy and support URLs for released apps.
- Current featured product: the Usage iOS app.

## Experience

- Homepage sections: hero, work categories, current product, about, and footer.
- Dark/light theme switch stored in browser `localStorage`.
- Usage has dedicated privacy and support pages.
- Contact address is published on the site and in support/policy content.

## Trust boundaries

- No backend, account system, analytics, advertising, or build framework is present
  in this repository.
- Public legal/support wording must stay aligned with actual app behavior.
- Do not publish private drafts, credentials, or personal data that is not already
  intentionally public.
